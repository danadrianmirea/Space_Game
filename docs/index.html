A game like "Chicken Invaders", developed in pygame (a python package)
This was my first experience in using pygame!

How to play

  • First you need to download the game codes from GitHub, Windows executable version will be available soon!

  • Keyboard: you can use left and right arrow keys for movement and space button for shooting
  • Mouse: you can also use mouse motion for movement and left mouse button for shooting
